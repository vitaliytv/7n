## ADR Фільтрація шумних шляхів із diff-контексту для LLM-агента у команді `push`

**Status:** Accepted
**Date:** 2026-06-03

## Context and Problem Statement
При генерації коміт-меседжу LLM-агент отримував повний `git diff --cached`, який містив шумні файли (ADR, CHANGELOG, lock-файли, артефакти збірки). Це заважало агенту визначити суть коміту — другорядні зміни заглушували значущі. Пізніше вимога розширилась: виключати весь `docs/` (не лише `docs/adr/`), а в stdout замість поштучного переліку ADR-файлів показувати лише їх кількість.

## Considered Options
* Виключати вміст шумних шляхів із diff через git pathspec `:(exclude)` — передавати агенту diff без їхнього вмісту, але зберігати повний перелік файлів (`--name-status`) для scope
* Інші варіанти в transcript не обговорювалися.

## Decision Outcome
Chosen option: "Виключати вміст шумних шляхів через `:(exclude)` pathspec", because користувач явно попросив зменшити шум для визначення суті коміту, зберігши перелік файлів для scope.

### Consequences
* Good, because transcript фіксує очікувану користь: агент бачить повний перелік файлів (`--name-status`) для визначення scope (workspaces), але diff без вмісту `docs/`, CHANGELOG, lock-файлів та артефактів; перевірено на реальному diff — 0 `diff --git … docs/adr/` заголовків у відфільтрованому виводі, але всі 10 ADR присутні в `--name-status`.
* Bad, because `**/docs/**` не ловить кореневий `docs/` — потрібні обидва патерни `docs/**` і `**/docs/**`; transcript фіксує цей факт як нетривіальну деталь реалізації.

## More Information
Дефолтний набір виключень: `docs/**`, `**/docs/**`, `**/CHANGELOG.md`, `**/.changes/**`, `*.lock`, `package-lock.json`, `pnpm-lock.yaml`, `yarn.lock`, `**/*.d.ts`, `**/*.snap`, `**/__snapshots__/**`, `**/*.min.js`, `**/*.map`, `dist/**`, `build/**`, `coverage/**`. Конфігурація через env: `N7COMMIT_NO_DEFAULT_EXCLUDE=1` (вимкнути дефолти), `N7COMMIT_EXCLUDE` (додати свої glob-и), `N7COMMIT_MAX_DIFF_DIFF_LINES` (ліміт рядків, дефолт 1500). У stdout `docs/adr/`-файли не перелічуються поштучно — друкується один рядок `📄 docs/adr/: N файл(ів)`. Реалізація і тести: `npm/push.js`, `npm/tests/push.test.mjs`.
