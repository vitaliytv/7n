## Проблема з «виносом»

Якщо «вивести git-логіку з @nitra/cursor» — означає Тип 1: це **неможливо без кругової залежності**.

```
@7n/n  ──depends on──▶  @nitra/cursor
(якщо @nitra/cursor тягне @7n/n — коло)
```

`dispatcher`, `sync-rules`, `branch` обслуговують **сам `@nitra/cursor`**, не `@7n/n`. Їх можна переписати з нуля в `@7n/n`, але це не «вивести» — це «продублювати для іншого контексту».
