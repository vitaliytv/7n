## 1. Як насправді запускати omlx

Це **LLM inference server**, не CLI-запускач. Два способи:

```bash
# Фоновий daemon (через launchd, Homebrew-style)
omlx start

# Або foreground (видно логи)
omlx serve --model-dir ~/models
```

Моделі завантажуються **автоматично** з піддиректорій `--model-dir`. Жодного `omlx run <model>` — я помилився у попередній відповіді, це аналогія з Ollama яка тут не діє.

UI доступний на `http://localhost:8000/admin/chat`, API — `http://localhost:8000/v1` (OpenAI-compatible).
